# Restaurant photos

Place your restaurant photos here. The app will use them for reviews.

**Naming convention:** Use these filenames to match reviews (in order):
- `photo1.jpg` → Paradise Biryani, Pune
- `photo2.jpg` → The Ivy, London
- `photo3.jpg` → Sukiyabashi Jiro, Tokyo
- `photo4.jpg` → Dishoom, London
- `photo5.jpg` → Trishna, Mumbai

Add more as `photo6.jpg`, `photo7.jpg`, etc. for other reviews in the feed.

Supported formats: .jpg, .jpeg, .png, .gif, .webp
