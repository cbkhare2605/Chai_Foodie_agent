# Foodie - Trusted Restaurant Reviews

A modern web app for restaurant reviews from your trusted connections. Rebuilt from the original Foodie app (Ionic 3 / Angular 5).

## Run the Website

```bash
cd foodie-website
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Demo Mode

The app runs in demo mode—no backend required. Click "Sign up / Login" to enter, then explore Feed, Add Review, and Map views.

## Features

- **Feed** – Reviews from your network (connections & friends of friends)
- **Add Review** – Search restaurants, rate 1–5, add wheelchair accessibility
- **Map View** – Restaurants by location (demo shows a list)
- **Profile** – Edit profile, My Reviews, Feedback
- **Trust levels** – Level 1 (direct), Level 2 (friends of friends), Level 3 (your reviews)
