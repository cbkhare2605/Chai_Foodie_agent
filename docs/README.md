## How to use this index

1. **Before creating a doc:** Check this index and `docs/sops/SOP-document-creation.md` so new docs follow the same structure and naming.
2. **After creating a doc:** Add it to the table above and set status (Draft ‚Üí In review ‚Üí Current).
3. **When changing a doc:** Update "Last updated" at the top of the doc and in this index if the change is significant.
